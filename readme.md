## Usage

```bash
$ npm install --global test-micro
$ test-micro
```


## Deployment

This microservice can be deployed to [now](https://zeit.co/now) by Zeit.
Assuming you've got `now` installed and set up:

```bash
$ now nhtera/test-micro
```
